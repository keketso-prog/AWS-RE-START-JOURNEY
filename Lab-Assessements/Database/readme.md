# Amazon RDS Hands-On Labs

Completed practical labs working with Amazon RDS to build and manage relational databases. These labs provided experience with database creation, connectivity, and operations.

# Labs Completed

# Build Your DB Server and Interact With Your DB Using an App
- Creating highly available RDS instances
- Configuring database settings and parameters
- Connecting RDS to web servers
- Testing database connectivity with applications

# RDS Database Operations
- Launching RDS database instances
- Connecting to databases from Linux servers
- Creating database tables and schemas
- Inserting and managing data
- Performing SQL queries and operations

## Key Skills

Gained hands-on experience deploying and managing relational databases on AWS. Learned to configure RDS instances for high availability, establish secure connections, and perform standard database operations using SQL.
